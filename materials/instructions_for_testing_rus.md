# Инструкция по запуску тестов

Помимо тестов на корректные выходные данные система автотестирования будет
проверять твою программу и ее исходный код по следующим пунктам:

* **Стилевые тесты.** При написании кода на _C_ и _C++_ необходимо придерживаться _Google C++ Style Guide_. Чтобы проверить, насколько красота твоего кода соответствует стандартам, ты можешь протестировать свой код с помощью утилиты _clang-format_. В папке ```materials/linters``` лежит файл ```.clang-format```, который содержит необходимые настройки для стилевого теста. Данный конфигурационный файл распространяет свое действие на все файлы, которые лежат с ним в директории или в директориях ниже. Поэтому, чтобы данные настройки применились к твоим файлам с исходным кодом, скопируй ```.clang-format``` в папку src. \
  \
  Чтобы запустить проверку на стиль, выполни следующую команду: \
  ```clang-format -n src/sourcefile_name.c``` 

  Необходимая версия clang-format: \
  **Mac** 18.1.8 \
  **Linux** 18.1.8
